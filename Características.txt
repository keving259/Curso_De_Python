Lenguaje de programación de propósito general: Puede ser usado para cualquier actividad (no está diseñado para algo específico).
Lenguaje de alto nivel: Se parece al lenguaje natural.
Fácil de aprender: Alta curva de aprendizaje.
Lenguaje de tipado dinámico: A diferencia del estático, Python identifica los tipos de datos sin necesidad de especificarlo explícitamente.
Orientado a objetos.
Lenguaje interpretado: A diferencia del compilado, hay un programa que interpreta instrucción por instrucción y las va ejecutando (lo convierte a lenguaje máquina). Suelen ser más lentos, porque interpreta y ejecuta, el compilado solo ejecuta.

Permite a los desarrolladores resumir mucho código y ordenar mejor el código.
Cuenta con una gran biblioteca estándar de código.
Se puede mezclar con otros lenguajes.
Comunidad grande.
Multiparadigma: Se puede pasar a diferentes SO y sigue funcionando.


Se puede usar para:
Desarrollo web del lado del servidor.
Aplicaciones móviles.
Automatización de scripts.
Ver archivos Log4J: Biblioteca con un error de seguridad que se debe actualizar.
Machine Learning.
Análisis de datos.
Desarrollo de Software.
Desarrollo de juegos.
Hacking.


No se puede acceder a registros de hardware, no se pueden crear drivers, ni programar microcontroladores de forma nativa.


Historia de Python.
Guido van Rossum es el creador de Python. Lanzó la versión 0.9.0 en 1991, ya incluía tipos e datos y funciones para solucionar errores. Su filosofía siempre fue crear un lenguaje fácil de aprender y de escribir, pero sin frenar su potencial. Python 1 se lanzó en el 94, con nuevas funciones para procesar listas de datos. Python 2 se lanzó el 16 de octubre del 2000, con nuevas características, como compatibilidad con carácter Unicode y formas más cortas de recorrer una lista.
El 3 de diciembre de 2008 salió la versión 3, con funciones de impresión y más soporte de errores.
